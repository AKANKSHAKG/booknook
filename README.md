# BookNook — Reading Companion

## APIs Used
- Google Books API → https://console.cloud.google.com (enable "Books API")
- Dictionary API → no key needed
- NewsAPI → https://newsapi.org
- Quotable → no key needed

## Setup
1. Copy `.env.example` → rename to `.env`
2. Fill in:
   REACT_APP_BOOKS_KEY=your_key
   REACT_APP_NEWS_KEY=your_key
3. npm install && npm start
